# Owner Discovery Checklist

Completează înainte de popularea finală și publicare. Câmpurile marcate critic blochează lansarea.

## A. Identitate și legal — critic

- [ ] Denumirea legală exactă
- [ ] Denumirea comercială
- [ ] Forma juridică și datele ce trebuie afișate
- [ ] Logo existent / nevoie de logo nou
- [ ] Domeniul dorit
- [ ] Persoana responsabilă pentru aprobări

## B. Servicii și operare — critic

- [ ] Lista exactă a serviciilor oferite
- [ ] Servicii neacceptate
- [ ] Zone/localități acoperite
- [ ] Program
- [ ] Timp mediu de răspuns
- [ ] Lucrări urgente: condiții
- [ ] Achiziționare materiale: da/nu și condiții
- [ ] Număr real de membri ai echipei
- [ ] Experiență verificabilă
- [ ] Garanție: durată, scope, excluderi

## C. Preț și ofertare

- [ ] Se publică prețuri „de la”?
- [ ] Unități: m²/lucrare/zi
- [ ] Ce este inclus/exclus
- [ ] Factori de complexitate
- [ ] Deplasare și evaluare
- [ ] Buget minim de proiect, dacă există
- [ ] Regula estimatorului orientativ

## D. Portofoliu — critic

Pentru fiecare proiect:

- [ ] titlu intern
- [ ] localitate generală
- [ ] tip lucrare
- [ ] servicii
- [ ] dată/durată/suprafață, dacă se publică
- [ ] challenge/solution/result
- [ ] before/during/after
- [ ] drept de publicare
- [ ] testimonial asociat
- [ ] date de identificare ce trebuie eliminate

## E. Canale — critic

- [ ] Telefon final
- [ ] Viber
- [ ] Telegram
- [ ] Email
- [ ] Instagram
- [ ] TikTok
- [ ] Facebook
- [ ] Google Business Profile existent
- [ ] Conturi/directoare existente

## F. Conținut și limbi

- [ ] RO la lansare
- [ ] RU la lansare
- [ ] EN ulterior
- [ ] Persoană care verifică limba rusă
- [ ] întrebări frecvente reale
- [ ] recenzii și surse
- [ ] fotografii cu echipa/procesul

## G. Lead management

- [ ] Cine răspunde lead-urilor
- [ ] SLA dorit
- [ ] Canale preferate
- [ ] Statusurile pipeline acceptate
- [ ] Motive de pierdere
- [ ] Follow-up actual
- [ ] Telegram notifications da/nu
- [ ] Export/reporting necesar

## H. Buget și growth

- [ ] Buget foto
- [ ] Buget video/image sequence
- [ ] Buget 3D P2
- [ ] Buget lunar SEO
- [ ] Buget ads
- [ ] Buget call tracking
- [ ] Cine administrează site-ul

## Gate de finalizare

Nu se publică până când secțiunile A, B, D și E nu sunt complete. Funcțiile dependente de secțiunile incomplete se ascund sau rămân draft; nu se completează prin presupuneri.
