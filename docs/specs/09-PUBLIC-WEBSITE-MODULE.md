# Public Website Module

## Responsabilitate

Livrează homepage-ul, layout-ul global și paginile publice generale: Despre, Proces, Recenzii, FAQ, Prețuri, Contact și paginile juridice.

## Homepage section contracts

| Secțiune | Scop | Date necesare | CTA |
|---|---|---|---|
| Hero | promisiune + localizare | headline, media, service area | Estimare / Proiecte |
| Trust strip | reducere risc | ani, garanție, echipă, arie — confirmate | Proces |
| Servicii | orientare | servicii prioritare și beneficii | Vezi serviciul |
| Flagship project | dovadă | proiect complet cu media | Vezi proiectul |
| Precision proof | diferențiere | macro detalii și explicații | Standardul nostru |
| Process | predictibilitate | etape reale | Programează evaluare |
| Estimator teaser | calificare | reguli estimator | Calculează orientativ |
| Reviews | social proof | recenzii cu sursă | Toate recenziile |
| FAQ | eliminare obiecții | întrebări reale | Contact |
| Final CTA | conversie | canale și program | Sună / Trimite cerere |

## Pagina Despre

- poveste reală și experiență;
- cine execută lucrările;
- aria și tipurile de proiecte;
- standarde de lucru;
- fără fotografii stock de „echipă”.

## Pagina Proces

Etape recomandate:

1. cerere și imagini;
2. calificare;
3. evaluare la locație, dacă este necesară;
4. ofertă și scope;
5. pregătire/protecție;
6. execuție și verificări;
7. predare și recomandări de întreținere;
8. garanție conform condițiilor confirmate.

## Prețuri

- publică „de la” doar cu aprobare;
- explică unitatea și ce este inclus;
- listează factorii de complexitate;
- afișează data actualizării;
- include disclaimer că evaluarea finală depinde de suprafață și condiții.

## Recenzii

- sursă, dată și proiect asociat unde este posibil;
- nu publica rating agregat fără date eligibile;
- administrare de consimțământ;
- fără recenzii inventate sau rescrise într-un mod care schimbă sensul.

## Contact

- telefon, Viber, Telegram, email și social confirmate;
- program și timp estimat de răspuns;
- hartă doar dacă adresa publică este legitimă; altfel service area;
- formular rapid;
- schema LocalBusiness/HomeAndConstructionBusiness.

## Empty/fallback states

- dacă nu există suficiente proiecte, nu se afișează carusel gol;
- dacă o recenzie nu are sursă, rămâne draft;
- dacă media hero eșuează, posterul rămâne complet;
- dacă un canal de contact lipsește, nu se afișează icon inert.

## Acceptance criteria

- mesaj și CTA în primul viewport pe mobil;
- datele NAP sunt consistente;
- toate secțiunile au conținut real sau sunt ascunse;
- fără CLS din imagini/fonturi;
- toate CTA-urile au analytics și context.
