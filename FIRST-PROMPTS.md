# Primele prompturi recomandate

## Cowork — prima sarcină

Analizează directorul proiectului fără să modifici cod sau documente. Citește `CLAUDE.md`, `docs/specs/00-README.md`, `docs/specs/26-OWNER-DISCOVERY-CHECKLIST.md` și `docs/work/STATUS.md`. Verifică dacă structura proiectului respectă `SETUP-GUIDE.md`. Livrează: lipsuri, conflicte de referințe, informații `CONFIRM_OWNER`, ordine recomandată pentru primul milestone și un handoff complet pentru Claude Code. Nu începe implementarea.

## Claude Code — verificarea configurației

Lucrează numai în Plan mode. Verifică ce instrucțiuni, rules și settings sunt încărcate. Citește `CLAUDE.md`, `docs/specs/00-README.md`, `docs/specs/05-ARCHITECTURE-DECISION-RECORDS.md`, `docs/specs/28-MASTER-IMPLEMENTATION-PLAN.md` și `docs/work/STATUS.md`. Nu modifica nimic. Raportează structura detectată, conflictele, lipsurile, comenzile care ar trebui definite după inițializarea proiectului și propune primul task implementabil după Gate A.
